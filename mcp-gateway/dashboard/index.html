Setup Guide
1
Install cloudflared
brew install cloudflare/cloudflare/cloudflared
2
Configure tunnel domain in the WhatsApp tab, then click "Start Tunnel" above

With your own domain the URL is stable; without one a random URL is assigned per session.

3
Register the webhook in Meta
  1. Meta Developer Console → your app → WhatsApp → Configuration
  2. Webhook → Edit → paste the Webhook URL (from WhatsApp tab)
  3. Enter your Verify Token
  4. Subscribe to messages field → Verify and Save
Gateway API Token

This token authenticates the daily briefing dashboard with the gateway. Copy the GATEWAY_API_TOKEN line into daily-briefing-dashboard/.env.

After rotating, update GATEWAY_API_TOKEN in daily-briefing-dashboard/.env and restart the dashboard server.
